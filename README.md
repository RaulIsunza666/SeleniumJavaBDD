# Selenium Automation Framework

Automation framework developed with:

- Java 21
- Maven
- Selenium 4
- TestNG
- Selenium Manager

## Architecture

- Page Object Model
- Driver Factory
- Base Page
- Base Test
- Configuration Reader

## Test Execution

```bash
mvn clean test
```

## Open Test Report

```bash
allure serve target/allure-results
```
