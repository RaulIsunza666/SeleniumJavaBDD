# Selenium Automation Framework

Automation framework developed with:

- Java 21
- Maven
- Selenium 4
- TestNG
- Selenium Manager
- Allure

## Architecture

- TestNG
- Selenium
- DriverFactory with ThreadLocal
- Page Object Model
- Logs
- Allure
- Cucumber
- Hooks
- HTML report for both scenarios (TestNG/Cucumber)

## Test Execution

```bash
mvn clean test
```

## Open Test Report

```bash
allure serve target/allure-results
```
